# Opacity Negotiation Demo

텍스트 에디터에서 AI 증강 기능을 구현한 데모 프로젝트입니다. 사용자가 텍스트를 선택하면 AI 증강 버튼이 나타나고, 클릭 시 선택한 텍스트 뒤에 AI가 생성한 텍스트가 추가됩니다.

## 주요 기능

- 텍스트 선택 시 AI 증강 버튼 표시
- AI 생성 텍스트의 점진적 불투명도 조정
- 실시간 텍스트 편집

## 설치 방법

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

## 기술 스택

- React
- TypeScript
- Tailwind CSS
